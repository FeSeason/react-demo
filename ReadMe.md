脚手架只是搭设的一个简易版本可以快速启用，目的的为了能够快速上手，*React、React-Router*，以及*Redux*
---------------------------------------------------------------

**安装：** npm install

**开发版本：** npm run build

**生产版本：** npm run dev

**默认端口：** 8888 [http://localhost:8888](http://localhost:8888)




> 主要功能

- 用户列表
- 查看详情

```
涉及功能点：

* 路由切换
* 路由传参
* 组件嵌套
* 父子组件数据传递
* 子组件改变父组件状态
* 通过redux的数据传递（用户列表信息获取，以及路由切换后获取redux数据）
```
